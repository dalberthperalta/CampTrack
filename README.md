# CampTrack ⛺

## 📝 Descripción General del Proyecto
El proyecto consiste en el desarrollo de un **sistema de inventario para la gestión de equipos de camping extremo**. Permite controlar el registro, disponibilidad, préstamos y mantenimiento de los equipos. 

El sistema facilita la organización mediante categorías y el seguimiento del estado de cada elemento. Además, integra una base de datos para asegurar la información y optimizar la administración del inventario. Su objetivo es mejorar la eficiencia y el control de los recursos disponibles.

---

## ⚙️ Requerimientos Funcionales

* **📦 Gestión de Equipos:** Registrar, editar, eliminar y consultar equipos; búsqueda por nombre, categoría o estado.
* **📂 Gestión de Categorías:** Crear, editar, eliminar y asignar categorías a los equipos.
* **🤝 Control de Préstamos:** Registrar préstamos, validar disponibilidad, gestionar devoluciones e historial.
* **🛠️ Registro de Revisiones y Mantenimiento:** Registrar estado, daños y mantenimiento de equipos.
* **📊 Control de Inventario:** Mostrar disponibilidad, actualizar stock automáticamente y alertar bajo inventario.
* **⚠️ Gestión de Daños y Reposición:** Registrar daños, equipos fuera de servicio y reposiciones.

---

## 🚀 Requerimientos No Funcionales

* **🔒 Seguridad de los Datos:** Protección contra accesos no autorizados y respaldo de información.
* **⚡ Rendimiento del Sistema:** Respuestas rápidas y manejo eficiente de datos.
* **🎨 Facilidad de Uso:** Interfaz intuitiva y navegación sencilla.
* **📈 Escalabilidad:** Posibilidad de crecimiento y ampliación del sistema.
* **🏗️ Mantenimiento:** Código organizado con separación de capas.
* **✅ Disponibilidad:** Acceso confiable y continuo al sistema.
